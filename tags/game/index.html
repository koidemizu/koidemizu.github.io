Game

Stop_and_Go

ワンボタンゲーム。
ボタンが押されている間は自機の進行が止まります。
敵の弾をよけながらゴールを目指します。弾の色によって挙動が違うので注意してください。
クリアするとだんだんと敵の攻撃力とゲームスピードがアップしていきます。
(キーボード ・スマホ対応)

Action-Shooting

2Dアクション。 各ステージでターゲットを破壊します。
(キーボード・バーチャルパッド対応)

Maze

迷路を探索するゲーム。
懐かしいワイヤーフレームのグラフィックを目指しています。
プレイヤーは敵キャラクターやギミックに対応しながらゴールを目指します。 (キーボード ・バーチャルパッド対応)

Action-RPG1

シンプルなレトロアクションRPG。
プレイヤーは様々なアイテムを探しながら、各チャプターごとの目標達成を目指します。
(キーボードのみ対応)

Action-RPG2

クオータービューのレトロアクションRPG。
プレイヤーは地下ダンジョンに潜り、クエストをこなしながら探検をします。

Command-RPG

コマンド式のローグライクRPG。

Fanart-Bloodborne

Hanabi

Hieroglyph

日本語のひらがなをヒエログリフに変換するアプリ。

Oekaki
